Budget Line

A Narrative-Driven Financial Survival Simulation

📌 Overview

Budget Line is a serious game designed to simulate the lived experience of financial hardship. Players take on the role of an individual managing limited income while balancing essential needs such as rent, food, health, and wellbeing.

Through decision-making, time management, and unpredictable life events, the game explores the emotional and systemic pressures of living pay-cheque-to-pay-cheque.

This project was developed as part of a final-year university module, focusing on combining game design, simulation systems, and social impact storytelling.

🕹️ Player Interaction

Players interact with objects in the environment:

Object	Action
🛏 Bed	Restore energy
📚 Desk	Study (increase grades)
🍗 Fridge	Eat (costs money, restores hunger)
🚿 Sink/Shower	Improve hygiene
🚪 Door	Travel / Work / University

Each action:

Consumes time
Affects stats
Can trigger events

🏗️ System Architecture

The game follows a modular system architecture:

GameState System → Manages all player stats
Time System → Controls progression of day cycles
Event Manager → Triggers random and scripted events
Player Actions → Handles interactions and consequences
UI System → Displays stats, menus, and event popups
